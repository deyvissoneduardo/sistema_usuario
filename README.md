# sistema_usuario
Sistema de Usuarios com banco de dados .txt<br>
Com fins de aprimiror estudos na area de shell script. Para utiliza - lo <br>
em seu terminal clone o projeto com o seguinte comando 
```
https://github.com/deyvissoneduardo/sistema_usuario.git
```
ou baixe o arquivo .zip e descompact. Na raiz do projeto excute o comando
```
source ./sistema_de_usuarios.sh 
```
Este comando ira carrega as funcionalidades implementadas para o bash atual.
Caso fecher o terminal, ser√° necessario executa o comando novamente.
Apos isso estara disponivel em seu terminal todas funcionalidas.
Para usar pasta chamar passando os parametros necessarios
```
ListUser 
```
Lista todos Usuarios.
```
InsertUser id:nome:email 
```
Ira inserir um usuario
```
DeleteUser nome 
```
deleta o usuario selecionado.
